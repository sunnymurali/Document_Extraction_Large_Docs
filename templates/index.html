API Documentation

POST /api/extract

Upload a PDF and extract structured data.

Request
  • file: PDF file (multipart/form-data)
  • extractionSchema: Optional JSON schema (form field)
Response
{
  "success": true,
  "data": {
    // Extracted structured data
  }
}
Example Schema
{
  "fields": [
    {"name": "invoice_number", "type": "string", "description": "The invoice number"},
    {"name": "date", "type": "date", "description": "The invoice date"},
    {"name": "total_amount", "type": "number", "description": "The total invoice amount"}
  ]
}

How it works
  1. Upload your PDF document
  2. Optionally provide a schema to guide extraction
  3. Our API extracts text from the PDF
  4. OpenAI analyzes the content and structures the data
  5. Receive structured JSON data in response
