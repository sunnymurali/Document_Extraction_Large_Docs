API Documentation

This tool provides a REST API to extract data from PDFs:

POST /api/extract
Content-Type: multipart/form-data

file: [PDF File]
extraction_schema: (optional) JSON schema

Example schema:

{
  "fields": [
    {"name": "invoice_number"},
    {"name": "date"},
    {"name": "total_amount"}
  ]
}
